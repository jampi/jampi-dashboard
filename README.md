# Jampi Dashboard
Web interface for Jampi backend.

## Requirements
* Python 3.x (with ```virtualenv``` and ```pip```)

## Install
1. ```git clone git@github.com:jampi/jampi-dashboard.git```
2. ```cd jampi-dashboard```
3. ```virtualenv3 env```
4. ```source env/bin/activate```
5. ```pip install -r requirements.txt```
6. ```python app.py```
